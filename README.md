# Anime Orbit

Anime Orbit is a Chrome extension for anime fans who want one place to discover
official streaming platforms, check the daily airing schedule, follow latest
releases, and read anime news.

## GitHub Pages

This repository also includes a small GitHub Pages site for the product home
page and privacy policy.

- Homepage: `https://KrishnaPapana.github.io/anime-orbit/`
- Privacy Policy: `https://KrishnaPapana.github.io/anime-orbit/privacy-policy.html`
