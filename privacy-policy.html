Anime Orbit

Privacy Policy

Last updated: April 20, 2026

Anime Orbit is a Chrome extension that helps users discover official anime streaming platforms and view public anime schedule, release, and news information.

Information Anime Orbit collects

Anime Orbit does not collect personal account information, payment information, passwords, or browsing history.

Anime Orbit stores one local preference in Chrome storage:

  • Selected region filter

Information Anime Orbit accesses

Anime Orbit requests access to the following public data sources in order to provide its features:

  • https://graphql.anilist.co/*
  • https://news.google.com/*

How data is used

Anime Orbit uses data only for the extension's single purpose:

  • showing official anime platforms
  • showing anime airing schedules and release information
  • showing anime news headlines
  • remembering the user's selected region preference

Data sharing

Anime Orbit does not share personal user data with third parties.

Data retention

The locally stored region preference remains in Chrome storage until the user clears extension data, uninstalls the extension, or changes the saved preference.

Security

Anime Orbit is designed to use the minimum permissions needed for its functionality. It does not inject remote code and does not require user login.
